# VIVA - 놀이공원 예약 관리 시스템

놀이공원 예약을 오프라인에서 온라인으로 전환하기 위한 JSP 기반 웹 애플리케이션입니다.
Model 1 방식으로 개발되었으며, 고객은 온라인으로 예약하고 관리자는 예약 현황과
어트랙션을 한 화면에서 조회·관리할 수 있습니다.

> 본 프로젝트는 2026년 1월~2월, 6인 팀 프로젝트로 진행되었으며,
> 본 저장소는 프로젝트 종료 후 개인 학습 아카이빙 목적으로 재구성하여 업로드한 것입니다.

## 기술 스택
- Java, JSP/Servlet (Model 1)
- Oracle Database
- HTML/CSS/JavaScript, jQuery

## 담당 역할 (박제영)
관리자 기능 전체를 담당했습니다.

- **회원 관리** (`userinfoAdmin`) — 회원 정보 조회 및 관리
- **예약 현황 관리** (`reserveAdmin`) — 예약 조회·수정·취소 처리
- **어트랙션 관리** (`attractionAdmin`) — 놀이기구 정보 CRUD
- **공지사항 관리** (`noticeAdmin`) — 공지 등록·수정·삭제
- **1:1 문의 관리** (`qnaAdmin`) — 문의 조회 및 답변
- **공연/행사 관리** (`perfAdmin`) — 공연·행사 정보 CRUD
- **로그인/인증** (`loginAdmin`) — 관리자 로그인 처리
- **대시보드** (`dashboardAdmin`) — 관리자 메인 대시보드
